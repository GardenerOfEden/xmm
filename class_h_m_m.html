Detailed Description

Hidden Markov Model.

Support Hierarchical Model: if built with the flag 'HIERARCHICAL', the model includes exit transition probabilities. The model can be eith unimodal, or Multimodal when constructed with the flag 'BIMODAL'.

void HMM::updateAlphaWindow ( )
protected

Estimates the likeliest state and compute the bounds of the windows over the states.

The window is centered around the likeliest state, and its size is the number of states. The window is clipped to the first and last states.
