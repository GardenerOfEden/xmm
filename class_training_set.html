void TrainingSet::recordPhrase ( int  phraseIndex,
vector< float > const &  observation 
)

Record training data.

The method appends an observation to the data phrase. The observation need to have a size "dimension". In bimodal mode, the observation must concatenate input and output observations. A phrase is created if it does not exists at the given index

Parameters
phraseIndexindex of the data phrase in the trainingSet
observationobservation vector to append to the phrase
Exceptions
runtime_errprif phrase has shared memory (construction with SHARED_MEMORY flag)
