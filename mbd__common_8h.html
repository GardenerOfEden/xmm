enum FLAGS

Flags for the construction of data phrases and models.

Enumerator
NONE 

no specific Flag: Unimodal and own Memory

SHARED_MEMORY 

Defines a shared memory phrase.

If this flag is used, data can only be passed by pointer. Recording functions are disabled and the memory cannot be freed from a Phrase Object.

BIMODAL 

Defines is a phrase is used to store bimodal data.

If this falg is used, the phrase contains 2 arrays for input and output modalities

HIERARCHICAL 

Defines is the model is used by a hierarchical Algorithm.

If this falg is used, the phrase contains 2 arrays for input and output modalities
