Detailed Description

Data phrase.

The Phrase class can be used to store unimodal and Bimodal data phrases. It can have an autonomous memory, or this memory can be shared with another data container. This can be specified by using the 'SHARED_MEMORY' flag in the constructor. The phrase can be either unimodal - i.e. it contains a single 2D array to store the data, - or bimodal - i.e. it contains 2 arrays to store the input and output modalities. The latter case can be specified by using the 'BIMODAL' flag in the constructor.
